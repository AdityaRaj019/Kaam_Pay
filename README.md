# Next.js Application

This project is developed using [Next.js](https://nextjs.org/) and was initialized with the `create-next-app` setup.

## Getting Started

Run the development server using one of the following commands:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

After starting the server, open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## Development

The main application page can be edited by modifying:

```bash
app/page.tsx
```

Changes are reflected automatically during development through hot reloading.

This project also uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) for optimized font loading and includes the [Geist](https://vercel.com/font) font family.
